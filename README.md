# labirynt

> work in progress

Fan-made digital reconstruction of _Labirynt Śmierci_, an unauthorized Polish edition of _Citadel of Blood_ released in the 1980s by the now-defunct company Encore, during a period when Poland lacked modern copyright law. Includes a LaTeX typeset rulebook and recreated game components (board tiles and tokens) - all in printer-friendly formats.
