# labirynt

> work in progress

Fan-made digital reconstruction of _Labirynt Śmierci_ board game released in the 1980s by the now-defunct company Encore.
_Labirynt śmierci_ was an unauthorized Polish edition of _Citadel of Blood_, published during a period when Poland lacked modern copyright law.
Includes a LaTeX typeset rulebook and recreated game components (board tiles and tokens) - all in printer-friendly formats.

## Rulebook

The rulebook was rewritten using LaTeX with modern Polish typography.
I dropped the three-column layout in favor of single-column, as it is easier to read.
Subsection enumeration was unified and titles were added, as the original had none.
The text is a faithful transcription of the original, with corrections limited to:
* spelling mistakes;
* typos;
* punctuation;
* linguistic errors;
* expanding abbreviations;
* obvious factual errors.
